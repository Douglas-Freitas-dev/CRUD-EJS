Bem-vindos ao meu repositório

Esse projeto tem como objetivo apresentar um CRUD simples de cadastro de produtos para lojas.
Para conseguir rodar com exatidão o projeto é preciso instalar algumas dependências, assim que fizer o clone do repositório siga os seguintes passos.

1 – Passo
Antes de tudo, instale o framework Node.JS em sua máquina para conseguir rodar a aplicação com exatidão.

2 – Passo
Após abrir o projeto no seu editor de código, inicialize um terminal e escreva os seguintes comandos, um após o outro.
npm i bootstrap

npm install ejs

npm install express

npm install express-ejs-layouts

npm install --save-dev nodemon

3 - Passo
Para inicializar o servidor e abrir a página web execute o comando 

nodemon index.js 

Pronto o programa estará em execução, abra uma página web e digite a url localhost:3000
